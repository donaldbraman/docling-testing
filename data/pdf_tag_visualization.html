About This Visualization

This report shows samples of tagged content from PDFs. Each color represents a different semantic tag type as defined in the PDF structure tree. This helps validate:

  • Coverage: Are important elements tagged?
  • Accuracy: Are tags semantically correct?
  • Completeness: Is most content tagged or are there gaps?
  • Tag types: What types of tags are used? Do they map to our schema?

Generated from PDF Structure Trees (StructTreeRoot) using pypdf library.
